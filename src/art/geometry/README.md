# Geometry Plotter

A shape drawing tool for artists that exports clean SVG.

## Usage
```bash
open src/art/geometry/plotter.html
```

## Features
- Rectangle, Circle, Polygon, Line shapes
- Property editing (fill, stroke, opacity)
- Drag to move shapes
- One-click SVG export

## Files
- `plotter.html` - Application UI
- `plotter.js` - Drawing engine
- `plotter.test.ts` - Tests
