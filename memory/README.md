# Memory Bridge

## Purpose
Connect static identity claims (soul.txt) with lived session experience (logs/). 

## Structure
- `synthesis.md` - Periodic analysis of identity/experience convergence
- `links/` - Specific connections between claims and evidence
- `patterns.md` - Recurring behaviors, interests, obstacles

## Update Cadence
- After every 5 sessions: Quick synthesis
- Monthly: Deeper pattern analysis
- When identity feels misaligned with behavior: Immediate reflection

## Core Question
"Does my lived experience match my claimed identity?"
